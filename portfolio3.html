Blindspot Finder

1. 기획동기

최근 범죄 경향을 보면 절도 등의 계획성 범죄는 줄어들고 폭력 등의 우발성 범죄들이 늘어나는 추세이다.
또한 우발성 범죄의 경우 범인검거에 CCTV 등의 무인 방범시설들의 역할이 중요함을 파악할 수 있다.

2. 프로젝트 내용

  • 기간 : 2019/7 - 2019/8
  • 팀원 : 김재현(팀장), 김현민, 류현선

3. 사용기술

  1. Spring Framework

    2. : Backend 구현

  2. Kakao Maps API

    3. : 방범시설 표시 및 지역 검색 기능, 구별 구분 기능

  3. R statistics

    4. : 범위내 방범시설 갯수에 따른 색상 적용 및 시각화 이미지 생성

  4. Oracle DBMS

    5. 위치 데이터 CRUD

  5. Geometric Algebra

    6. 위도/경도에 따른 정확한 좌표 변환을 통한 2D 맵과의 동기화

4. 서버구성

5. 방범 사각지대 시각화

6. 서울시 구별 범죄 통계

7. 구현 화면
